Contrato correspondiente a la entrega 1 de la microcredencial [“Tecnologías Blockchain, Contratos Inteligentes, Aplicaciones Descentralizadas y la Web 3.0” de la Universida de Oviedo](https://www.uniovi.es/en/estudia/titulospropios/microcredenciales/oferta/-/asset_publisher/NCtkAT4jPSMb/content/id/10937457)

## Cómo funciona

Primero es necesario desplegar este contrato en la red Blockchain mediante [Remix IDE](https://remix.ethereum.org/).
Después, se debe iniciar la [aplicación](https://github.com/alfonsodou/app-subasta) para poder interactuar con el contrato.
